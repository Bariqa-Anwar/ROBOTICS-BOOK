# Student Prerequisites

This document outlines the assumed prior knowledge for students engaging with the "Physical AI & Humanoid Robotics" book. The book does NOT include beginner tutorials for these topics.

## Assumed Prior Knowledge

Students are expected to have a basic proficiency in the following areas:

-   **Python Programming**: Fundamental concepts including variables, data types, control structures (if/else, loops), functions, and basic object-oriented programming.
-   **Linux Command Line**: Navigating the filesystem, basic file operations (copy, move, delete), executing commands, and managing processes.
-   **Git Version Control**: Basic operations such as cloning repositories, committing changes, branching, merging, and resolving simple conflicts.
-   **Linear Algebra**: Fundamental understanding of vectors, matrices, and basic transformations, as applied in robotics and 3D space.
